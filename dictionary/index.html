Типы данных

Типы данных:

  • Числа 
    var number_variable  = 10;
    Передаются по значению при копировании переменных
  • Строки 
    var string_variable  = "я строка";
    Передаются по значению при копировании переменных
  • Булевые или логические 
    var boolean_variable = true;
    Могут иметь только два значения true или false.
  • null 
    var null_variable = null;
    Обозначает что переменная пуста.
  • Объекты 
    var object_variable  = {};
    При копировании в переменные передается ссылка на объект. Объект при этом в памяти не копируется.

Операторы. Приведение типов. Условия. Switch
  • Операторы 
    || - Логическое ИЛИ. Срабатывает и возращает true если левая или правая часть true
== - Оператора НЕстрогого равенства. При сравнении конвертирует переменные разных типов в число.
=== - Оператора СТРОГОГО равенства. При сравнении не ковертирует.
&& - Оператор И . Срабатывает и возращает true если левая И правая часть true выражения true
! - Оператор отрицания. Приводит переменную к булевому значению и меняет на обратную.
>, < - Операторы сравнения БОЛЬШЕ, МЕНЬШЕ. Строки сравниваются не по длине, а по кодам букв в таблице.
>=, <= - Операторы сравнения БОЛЬШЕ ИЛИ РАВНО, МЕНЬШЕ ИЛИ РАВНО.
  • Приведение типов. 
    Приведение к числу осуществляется приставкой знака + перед переменной. - +"string"
Приведение к булевому значению осуществляется приставкой !! к переменной - !!"some_string"
  • Условные переменные. Используются для написания логики программ и их ветвления. 
    if ( true ) {
  console.log( 'Оператор условия в действии' );
}

if ( "Some_string" ) {
   console.log('Some_string');
}
    В условных операторах любая непустая строка, число не 0 является true.
  • Switch - используется как замена if else . 
            switch (day) {
            case "Суббота":
                x = "Сегодня Суббота";
                break;

            case "Воскресенье":
                x = "Сегодня Воскресенье";
                break;

            default:
                x = "Сегодня будний день";
                break;
        }

      BOM, DOM

      BOM

      Это набор глобальных переменных дляполучения информации о браузере и текущем УРЛ
      • window - является главным объектом браузера. Содежит в себе все глобавльные переменные. Является мостом между JS и DOM 
        console.log(window);
      • window.screen или screen - Содержит в себе информацию о экране пользователя. 
        console.log('Текущий размер экрана: Высота: ' + window.screen.height + " Ширина: " + window.screen.width);
      • navigator - Содержит в себе информацию о браузере пользователя. 
        console.log('Текущая информация о браузере: ' + navigator.userAgent);
console.log('ТВаша платформа: ' +  window.navigator.platform);
      • location - Содержит в себе всю информацию о текущем УРЛ страницы. Имеет методы для перехода на другую страницу или перезагрузки текущей. 
        console.log(location.href);
console.log('Текущий хост: ' + location.host);
console.log('Текущий путь: ' +  window.location.pathname);
console.log('Текущий протокол: ' +  window.location.protocol);
window.location.reload(); - метод для перезагрузки текущей страницы.

      DOM

      Объектная модель браузера отвечающая за управление HTML. Каждый HTML тег имеет представление в ввиде объекта.
      • getElementById - Получение ссылки на объект тега через его аттрибут id 
        var idsElement = document.getElementById('content-id');
      • getElementsByClassName - Поиск елементов по их аттрибуту class 
        var classElements = document.getElementsByClassName('test-class');
        Возращает объект HtmlCollection. По поведению схожий с массивом, но также имеет метод index для доступа к елементам по индексу в коллекции
      • getElementsByTagName - Поиск елементов по их аттрибуту class 
            var tagElements = document.getElementsByTagName('p');
        Возращает объект HtmlCollection. По поведению схожий с массивом, но также имеет метод index для доступа к елементам по индексу в коллекции

      Свойство DOM объектов

      • childNodes - содержит в себе массив всех дочерних елементов на один уровень ниже. Включает в себя теги и текстовые узлы.
      • children - содержит в себе массив всех дочерних елементов на один уровень ниже. Включает в себя только теги.
      • firstChild - Содержит в себе ссылку на первый дочерний елемент уровнем ниже
      • lastChild - Содержит в себе ссылку на последний дочерний елемент уровнем ниже
      • parentNode - Содержит в себе ссылку на родительский елемент уровнем выше
      • nodeType - Содержит в себе ТИП узла. Числовое значение. "1" : "ELEMENT_NODE", "2" : "ATTRIBUTE_NODE", "3" : 'TEXT_NODE', "4" : "COMMENT_NODE"
      • tagName - Свойство доступное только у узлов отвечающих за теги
      • innerHTML - Свойство содержащее весь HTML который находится внутри тега в ввиде текста. Свойство доступное только у узлов отвечающих за теги
      • attributes - Свойство содержит в себе массив всех аттрибутов узла.
      • nextSibling - Содержит в себе ссылку на следующий узел в дереве на том же уровне что и текущий.
      • previousSibling - Содержит в себе ссылку на предыдущий узел в дереве на том же уровне что и текущий.
