Задание.

а)Создать одностраничный интернет магазин.



б)Требования к программной части.

Требования. Вся логика магазина должна быть содержатся в одном объекте. Объект корзина тоже должен быть свойством самого объекта магазин.
Каждый товар в дополнению к свойствам ЦЕНА, НАЗВАНИЯ должен содержать свойство КОЛЛИЧЕСТВО_НА_СКЛАДЕ. Пользователь не должен иметь возможности купить единиц товара больше чем есть на складе.

Все события должны навешиваться на элементы через деллегирование. 
Все новые элементы должны вставляться на страницу через createElement и по возможности использовать техники частого изменения DOM дерева.


в)Описание элементов интерфейса.

1. Список товаров интернет магазина. Должен выводится на екран сразу после показа страницы. 
HTML  магазина должен генерироватся в Javascript на основании содержимого массива доступных товаров.
При отображении товаров  - товары колличество которых равно 0 не должны иметь кнопки КУПИТЬ(2).

2. Кнопка купить - после клика на нее товар должен добавлятся в корзину.

3. Блок краткой информации. При изменении колличества товаров в корзине - данные в этом блоке должны обновлятся.

4. Блок Корзина. При старте не должна быть видна. Отображать корзину только в том случае если есть в ней товары.

5. Кнопка Изменить колличество - При клике - должено показыватся окно с возможностью ввести колличество. При вводе 0 товар должен удалятся из корзины.

6. Кнопка Удалить - При клике - Товар должен удалятся из корзины.

7. Итоговая стоимость должна динамически менятся в зависимости от того, что происходит в корзине (Удаления, Изменения колличества)

8. Кнопка формить - При клике Должен отображаться блок 9.

9. Форма данных клиента - Набор инпутов для ввода информации о покупателе. Это форма при старте страницы не видна.

10. Кнопка Потвердить - При клике должно проверяться заполнены ли все поля. В обратном случае должна выводиться ошибка. В случае корректности данных - Должен отображаться элемент 11

11. Форма-Счет. Содержить все данные о заказе, список товаро, данные клиента и запись Спасибо за покупку. :-)
