Добработать задание Тест.

1. Взять или существующий или придумать самому тест. Сделать его осмысленным.

2. Добавить на старте input типа text для ввода имени человека котоырй начинает тест.
Не давать пользователю начать тест без введенного имени.

3. Добавить функционал который работает со свойством canGoBack объект Quiz .
Данное свойство отвечает за возможность возращаться к предыдущему вопросу.

4. Когда пользователь доходит до последнего вопроса и нажимает "Далее" - скрывать вопрос
и выводить на экран статистику (если ваш тест просто так, что нежелательно) или результат прохождения теста.

5. Сделать функционал прохождения теста заново. Разместить кнопку для начала прохождения заново ниже результатов.
Не забываем что для прохождения заново все значения которые ввел пользователь должны обнулиться. Текущий индекс вопроса тоже
должен вернуться в исходное состояние.

6. Результаты тестов хранить в новом  свойстве объекта Quiz в форме удобной для вас (массив, объект и т.д).
После нескольких прохождений теста должна
быть возможность вывести на экран результаты предыдущих прохождений.


============Задание для тех кто хочет посложнее===================
7. Расширить функционал чтобы вопросы содержали вариант ответа с именем "Ваш вариант ответа: ".
Для этого варианта овтета на экран должен выводится input type="text". Весь функционал должен сохранятся - сохранения введенного и т.д
